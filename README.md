## 🐍 The Snake Eats My Contributions

<div align="center">
<img src="https://raw.githubusercontent.com/allen745/allen745/output/github-contribution-grid-snake.svg" alt="snake animation"/>
</div>

<details>
<summary><b>⚙️ Not showing up yet? Click to activate it (one-time setup)</b></summary>

<br>

Needs a tiny GitHub Action on your `allen745/allen745` repo. First, enable **Settings → Actions → General → Workflow permissions → Read and write permissions**. Then add `.github/workflows/snake.yml`:

```yaml
name: Generate Snake
on:
  schedule:
    - cron: "0 0 * * *"
  workflow_dispatch:
  push:
    branches: [main]

jobs:
  generate:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk@v3
        with:
          github_user_name: allen745
          outputs: |
            dist/github-contribution-grid-snake.svg
      - uses: crazy-max/ghaction-github-pages@v4
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

Run it once manually from the Actions tab, and the snake will populate automatically after that.
